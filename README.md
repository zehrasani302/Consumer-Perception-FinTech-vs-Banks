# FinTech vs Banks Dashboard

## Project Overview

This Power BI dashboard analyzes survey responses collected during my ELP (Final Year Project) at IBA.

The project compares customer perceptions of traditional banks and FinTech services in Pakistan, highlighting trends in digital payment adoption, trust, and user preferences.

---

## Objectives

- Compare FinTech and traditional banking preferences
- Analyze demographic trends
- Study digital payment adoption like PayPak
- Generate actionable insights using survey data

---

## Tools Used

- Power BI
- Microsoft Excel
